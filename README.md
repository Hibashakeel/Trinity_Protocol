# TECHATHON - Futuristic Landing Page

A modern, immersive, and interactive landing page designed with a futuristic aesthetic. This project showcases advanced frontend techniques including custom cursor tracking, canvas animations, and responsive glassmorphism UI.

## ✨ Features

- **Interactive Custom Cursor:** A dual-element cursor (dot and ring) that reacts to movement with smooth interpolation and `mix-blend-mode` effects.
- **Dynamic Scroll Progress:** A multi-color gradient progress bar at the top that tracks reading depth.
- **Canvas Swirl Animations:** Real-time animated background elements using the HTML5 Canvas API for a high-tech feel.
- **Modern Typography:** Integrated with Google Fonts including 'Syne', 'Bebas Neue', and 'Space Mono'.
- **Responsive & Animated Sections:** Includes specialized areas for "The Vision," "Toolbox," and a unique "Floating Moments" interactive gallery.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom animations, Flexbox/Grid, and glassmorphism styling.
- **JavaScript (Vanilla):** Custom cursor logic, scroll listeners, and Canvas API for background effects.
